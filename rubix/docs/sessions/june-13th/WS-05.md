## Goal

The page-context/nav **backend**: a behaviour-affecting entity-tag store, a nestable org-scoped
navigation tree where each node assigns a board + context payload, and per-node access via the
existing grant model. This is what lets one board mount at many nodes, each parameterising and gating
it.

## Scope / deliverables

- **Entity-tag store** (new): org-scoped `(org, kind, entity_id, key, value)`; routes
  `PUT /api/v1/tags/{kind}/{id}` (full-replace), `GET /api/v1/tags/{kind}/{id}`,
  `GET /api/v1/tags/entities/{kind}` (reverse lookup), `GET /api/v1/tags/keys`. Because tags are now
  **behaviour-affecting**, write/read handlers resolve the target entity and enforce the **entity's
  own authz** (`edit` to write, `view`/read scope to read), and reject unknown/foreign ids. Sweep tags
  on entity delete (in the existing delete handlers).
- **`nav_nodes` table** via a forward-only `user_version` bump, dual-dialect, org-scoped, indexed on
  `(org, parent_id, sort_order)`. Shape: `id, org, parent_id?, title, sort_order, target jsonb,
  context? jsonb, icon?, accent?`. `target` is a tagged union — `{kind:"group"}` |
  `{kind:"dashboard", dashboard_id}` | `{kind:"route", route}`. The `route` value is a **closed
  allow-list** of built-in pages (sites|equips|points|dashboards|datasources|rules|boards|sparks|
  runs|audit|access). A `dashboard` target's id is validated **in the handler** to exist within the
  caller's **org** (a bare FK can't encode org-safety). On board delete, sweep dependent nodes to
  `{kind:"group"}`.
- **Nav CRUD + reorder/reparent** routes under `api/nav/**`; DTO registered; TS mirror authored.
- **`nav_node` grant kind:** register as grantable in the grant model. The nav-tree `GET` filters to
  nodes the principal holds `view` on (the existing per-node read-filter). Opening a node checks
  `view` on the **node**.
- **Default tree on org provision:** seed every static route as a node, granted org-wide, so gating is
  opt-in tightening — not a lockout. Static pages get access-gated for the first time.
